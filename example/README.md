### Description

Image Grid is plugin to select image from. gallery and gamera and set on grid easy to use;

**Code Sample**

      GridImage(
                compressImage: true,
                context: context,
                onchange: (List<File> files) {
                  setState(() {
                    imagecount = files.length;
                  });
                },
                title: "Select Image",
              ),
